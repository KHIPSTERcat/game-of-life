# Game Of Life

## Create solution
```
git clone https://github.com/KHIPSTERcat/game-of-life.git
cd game-of-life
pip install conan
conan remote add bincrafters https://api.bintray.com/conan/bincrafters/public-conan
mkdir build
cd build
conan install ..
cmake ..
```
